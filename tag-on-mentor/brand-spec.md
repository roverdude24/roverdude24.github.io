# Tag-Along Curriculum · Codex reference spec

System sentence: A black editorial canvas, oversized geometric sans type, quiet violet utility metadata, hairline rules, and occasional saturated visual fields turn the curriculum into a visual operating manual.

## Observed posture rules

1. Let the headline carry the page; supporting copy stays short and low-contrast.
2. Use black as the default surface, with white pages as deliberate breathers.
3. Keep navigation, volume labels, page numbers, and source metadata small, pill-shaped, and quiet.
4. Use hairline rules and open negative space instead of dense card chrome.
5. Reserve saturated color for one visual field or one typographic signal at a time.
